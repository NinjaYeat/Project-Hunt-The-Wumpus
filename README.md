Bonjour à tous,

Voici un projet réalisé dans le cadre de notre cours de systèmes informatiques. Le projet s'intitule Hunt-The-Wumpus.

Dans ce dépôt, vous pouvez consulter toutes les modifications ainsi que le code que nous avons développés afin de mener ce projet à bien.

Ce projet consiste à créer un jeu appelé Hunt-The-Wumpus.

Dans ce jeu, le joueur contrôle un personnage armé d'une flèche dont l'objectif est de chasser le Wumpus. Cependant, vous devrez être prudent lors de vos déplacements. Vous rencontrerez des flaques de sang, du slime et des chauves-souris. Bien qu'elles semblent inoffensives à première vue, elles peuvent indiquer la présence de dangers à proximité.

Certaines cases contiennent un puits mortel, tandis qu'une autre abrite le Wumpus lui-même. Si vous vous en approchez suffisamment, il vous tuera sans hésitation. De plus, une mauvaise utilisation ou un mauvais calcul lors du tir de votre flèche peut également entraîner votre mort.

Planifiez vos mouvements avec soin, restez vigilant, et surtout… amusez-vous !

Hello everyone,

This project was developed as part of our Computer Systems course. The project is called Hunt-The-Wumpus.

In this repository, you can find all the modifications as well as the code we developed to successfully complete this project.

This project consists of creating a game called Hunt-The-Wumpus.

In this game, the player controls a character armed with an arrow whose objective is to hunt the Wumpus. However, you must be careful when moving around the map. You will encounter blood stains, slime, and bats. Although they may seem harmless at first glance, they are often signs of nearby danger.

Some tiles contain deadly pits, while another hides the Wumpus itself. If you get too close, it will kill you instantly. Additionally, poor planning or incorrect calculation when shooting your arrow may also result in your death.

Plan your moves carefully, stay alert, and most importantly… have fun!

This project was developed by NinjaYeat (Amine Isenborghts) and Naiataria (Nora Lfarh).
